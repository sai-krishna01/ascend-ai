# MentorAI - AI-Powered Learning & Career Platform

Your personal AI mentor for academics, skills development, interview preparation, and career guidance. Available in English, Hindi & Telugu.

## Features

- **AI Teacher**: Learn concepts with patient, personalized explanations
- **Career Mentor**: Get career guidance and actionable advice
- **Mock Interviews**: Practice with realistic AI-powered interviews
- **Exam Mode**: Test your knowledge with adaptive questioning
- **Multi-lingual**: Support for English, Hindi, and Telugu
- **Progress Tracking**: Track your learning journey
- **Role-based Access**: Student, Teacher, Admin, and Founder roles

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, Framer Motion
- **Backend**: Supabase (Auth, Database, Edge Functions)
- **AI**: Integrated AI models for personalized learning
- **UI Components**: shadcn/ui

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Project Structure
